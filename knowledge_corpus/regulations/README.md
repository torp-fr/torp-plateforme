# Dossier — Textes réglementaires

Ce dossier contient les textes réglementaires applicables au secteur de la construction.

## Types de documents attendus

- **DTU** (Documents Techniques Unifiés) — normes d'exécution des travaux
- **Arrêtés ministériels** — textes officiels du Journal Officiel
- **Décrets** — textes de droit public relatifs à la construction
- **Réglementations thermiques** — RT 2012, RE 2020, etc.
- **Règlements de sécurité** — incendie (ERP), accessibilité (PMR), etc.
- **Textes normatifs légaux** — obligations des maîtres d'ouvrage et entreprises

## Formats acceptés

`.pdf` `.docx` `.txt` `.md`

## Nommage recommandé

```
DTU_26-1_enduits-mortier_2022.pdf
arrete_2023-01-15_performance-energetique.pdf
decret_n2021-872_chantiers-verts.pdf
```

## Notes

Les textes de ce dossier seront classifiés automatiquement comme `regulation`
et découpés selon les frontières d'articles (Article N / Art. N.M / Chapitre).
