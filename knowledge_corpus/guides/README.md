# Dossier — Guides techniques

Ce dossier contient les guides de mise en œuvre et bonnes pratiques du secteur BTP.

## Types de documents attendus

- **Guides de mise en œuvre** — méthodes d'application, techniques de pose
- **Fiches techniques fabricants** — prescriptions produits, conditions d'emploi
- **Guides de maintenance** — entretien préventif, périodicités d'intervention
- **Recommandations professionnelles** — OPPBTP, CSFE, SNBPE, etc.
- **Cahiers du CSTB** — études et recommandations techniques
- **Protocoles de réception** — essais, mesures, contrôles qualité

## Formats acceptés

`.pdf` `.docx` `.txt` `.md`

guide_mise-en-oeuvre_enduits-facade_v3.pdf
