# Dossier — Documentation technique générale

Ce dossier contient les documents techniques transverses qui ne rentrent pas dans
les catégories spécialisées ci-dessus.

## Types de documents attendus

- **Notices de calcul** — notes de dimensionnement structure, thermique, acoustique
- **Dossiers d'ouvrages exécutés (DOE)** — plans et notices de fin de chantier
- **Avis techniques (AT)** — agréments CSTB pour procédés non traditionnels
- **Cahiers des charges** — CCAP, CCTP généraux et particuliers
- **Mémoires techniques** — réponses à appels d'offres, justificatifs techniques
- **Plans et schémas textuels** — descriptions de plans, légendes, nomenclatures

## Formats acceptés

`.pdf` `.docx` `.xlsx` `.csv` `.txt` `.md`

## Nommage recommandé

```
DOE_batiment-A_lot03-electricite.pdf
avis-technique_AT20-17-234_isolation-ite.pdf
CCTP_lot01_gros-oeuvre_v2.docx
notice-calcul_structure-bois_EN1995.pdf
```

## Notes

Les documents de ce dossier seront classifiés par le pipeline automatique
(regulation / technical_guide / pricing_reference / jurisprudence / generic)
selon leur contenu. Ils constituent le dossier "fourre-tout" pour les documents
difficiles à classer a priori.
